# NuraLange

שיחה קולית אמיתית עם AI — מורה פרטי לשפות. React Native (Expo) ל-Web, iOS ו-Android.

## הרעיון

לא תרגילים ולא גיימיפיקציה מזויפת — **שיחה אמיתית** עם AI שזוכר שגיאות, מתאים קושי בזמן אמת, ומכניס מילים קשות לשיחה הבאה בצורה טבעית.

## תכונות

- **שיחה קולית AI-first** — Web Speech API בדפדפן, TTS בכל הפלטפורמות
- **חזרות חכמות** — אלגוריתם SM-2 בהשראת Duocards
- **8 תרחישים מהחיים** — מסעדה, שדה תעופה, ראיון עבודה ועוד
- **קושי דינמי** — התאמה בזמן אמת לפי הביצועים
- **ציון משמעותי** — שטף, דיוק, אוצר מילים, ביטחון, טבעיות
- **דוח שבועי** — התקדמות, שגיאות חוזרות, המלצה אישית
- **תמחור מדורג** — 5 תוכניות + חבילות תוספת (תשתית סליקה מוכנה)
- **עיצוב זכוכית** — mobile-first, RTL מלא בעברית

## התקנה

```bash
npm install
npm start
```

- `a` — Android
- `i` — iOS
- `w` — Web (זיהוי דיבור זמין)

## משתני סביבה (אופציונלי)

צור `.env` מהדוגמה:

```bash
cp .env.example .env
```

| משתנה | תיאור |
|--------|--------|
| `EXPO_PUBLIC_AI_API_URL` | כתובת API ל-AI (OpenAI/שרת עצמי) |
| `EXPO_PUBLIC_AI_API_KEY` | מפתח API |
| `EXPO_PUBLIC_PAYMENT_PROVIDER` | `mock` / `stripe` / `payplus` |
| `EXPO_PUBLIC_PAYMENT_API_URL` | שרת סליקה |

## מבנה הפרויקט

```
app/              — מסכים (Expo Router)
components/       — UI זכוכית, בוחר שפות
constants/        — עברית, תרחישים, ערכת נושא
lib/              — AI, דיבור, חזרות, תמחור, תשלומים
store/            — Zustand + AsyncStorage
```

## תמחור (לוגיקה)

| תוכנית | שיעורים/שבוע | מחיר/חודש | מחיר/שיעור |
|--------|-------------|-----------|------------|
| יומי | 5 | 200 ₪ | ~6.7 ₪ |
| 4/שבוע | 4 | 180 ₪ | 11.25 ₪ |
| 3/שבוע | 3 | 150 ₪ | 12.5 ₪ |
| 2/שבוע | 2 | 110 ₪ | 13.75 ₪ |
| 1/שבוע | 1 | 60 ₪ | 15 ₪ |

החבילה הגדולה ביותר תמיד הכי משתלמת — אין אפשרות "לרמות" את המערכת.
